e-store-express in NodeJS

  • Step 1: Scaffold a node app with express module. Serve the index page in the app folder.Test the server is listening on port 3000.
  • Step 2: Create the Authorization route on the root path. Also create the Authorization Controller. Test the GET /ping and POST /login route. To start with use the express features functions of Router() to return status of 200 and JSON object with status of running for /ping and status of login for /login. We will also scaffold function of ping and login in controller file in this step.
  • Step 3: Create a login schema with username, password as required properties. Add the middleware to validate the body of login route before calling into the controller method.
