Admin Beginner Trailhead

Admin Beginner

  • Card-1: Complete the following trail step: Salesforce Platform Basics. As you work through these steps, you will learn installing a dreamhouse package, adding a field to the contact object. You will understand that Salesforce platform address several usecases across departments of an enterprise. Salesforce Architecture is a cloud metadata driven SAAS offering backed with API, low code customizations, trust as a top priority and Saleforce Data + AI strategy is evolving the platform with the addition of Data Cloud and Einstein layers. You will also explore the setup menu and exchange.
  • Card-2: Complete the following trail steps: Data Modeling, Data Management. We know that we create standard and custom objects which defines that metadata and we understand that we can use features like Lookup and Master Detail to model relationships and there are point and click export and import utilities to load external data into objects and export data from objects in files.
  • Card-3: Complete the following trail step: Lightning Experience Customization. As we go through customizations like List Views, Compact Layout, Record Pages, Buttons and Links, we understand that we are designing a lightning apps with custom objects and lightning components. Custom actions further allow you to add features to lightning pages and apps with action type as create record, log call, flow, lightning component, email etc.
  • Card-4: Complete the following trail steps: User Engagement. As we go through User Engagement module, you will learn that there are tool tips like prompt and walkthrough that ease user onboarding, increase feature adoption and discovery, self service troubleshooting help and walkthroughs can be created for deeper learning.
  • Card-5: Complete the following trail step: Reports & Dashboards for Lightning Experience. As you work through these steps, we will learn how to choose an appropriate report type based on the requirements, apply filters and groups to meet the criteria for the records that need to be on the report. We can create dashboards with several components, reports to highlight metrics and trends with data in the org.
  • Card-6: About 10 days between Jan 1 to Jan 10
