estore-sfdc-system-api

Feature: Lead To Order E2E with Trail Automation and Mulesoft System API

Demo:Lead To Order E2E Demo

  1. Read through the trail Trail: Manage Products, Prices, Quotes, and Orders and start the estore-sfdc-system-api mule app github repository: estore-sfdc-system-api in the studio.
  2. Hit the ping uri and check for 200 OK.
  3. Query on lead John and Jane Doe with get on lead resource; as we don’t have them in the org, so we will create them in the next step.
  4. Create John Doe and Jane Doe with post on lead resource.
  5. We have products and pricebook from the trail in the system, so we have to delete them so that we can create them with the automation. We will query the pricebook id for Enterprise, Nonprofit and Standard Pricebook for Solar Panel, Sunpower and GreenSun. We will retrieve the pricebook Id and use the deletepricebookentry resource to delete pricebook entries. 
            
            select Id, Name, CurrencyIsoCode, UnitPrice, ProductCode from PricebookEntry where Pricebook2.Name = 'Standard Price Book' and ProductCode in ('SPC-R01', 'SPG-01', 'SPG-02')
  6. We will get on productsbyproductcodelist resource to retrieve the Ids for Solar Panel (SPC-R01), Sunpower (SPG-02) and Greensun (SPG-02). We will then use deleteproducts resource to delete the product entries.
  7. Do the get on productsbyproductcodelist resource again to make sure products are deleted and we receive empty list in the response.
  8. Create Solar Panel (SPC-R01), Sunpower (SPG-02) and Greensun (SPG-02) with post on product resource.
  9. Do the verification on Create and Customize Products and earn reward points.
  10. Create the Standard Pricebook entries for Solar Panel, then Sunpower and then GreenSun with post on pricebookentry resource.
  11. Create the Enterprise Pricebook entries as per trail spec for Japanese and Singapore currencies.
  12. Create the Nonprofit Pricebook entries as per the trail spec for british pound and us dollar.
  13. Do the verification on Create and Customize Products and earn reward points.
  14. John Doe works for Boy Scouts of the Southwest, so create the account and opportunity with post on convertlead resource. As per the spec, we have to pass the opportunity name as ‘BSofSW - Ground-mounted Panels’.
  15. Jane Doe works for Saito Family Laundromats, so create the account and opportunity with post on convertlead resource. As per the spec, we have to pass the opportunity name as ‘Saito - Ground-mounted Panels’.
  16. Update Boyscout opportunity with currency, amount and pricebook with put on opportunity/{id} resource.
  17. Add Greensun ground to the Boyscout opportunity with post on opportunity/product resource.
  18. Retrieve the account id for the Saito Family Laundromats account with get on account resource and passing account name as a query param.
  19. Update the Laundromat account with the account currency as Japanese with put on account/{id} resource.
  20. Update Laundromat opportunity with currency, amount and pricebook with put on opportunity/{id} resource.
  21. Add Sunpower and Greensun ground to the Laundromat opportunity with post on opportunity/product resource.
  22. Do the verification on Create an Opportunity and Add Products to It and earn reward points.
  23. Follow writeup for Control Access to Pricebooks and Products, do the verification and earn reward points. It is a one time setup.
  24. Crean a 80-cell quote for Saito - Ground-mounted Panels opportunity of Laundromat account with post on resource opportunity/quote resource.
  25. Crean a 60-cell quote for BSofSW - Ground-mounted Panels opportunity of Boy scout account with post on resource opportunity/quote resource.
  26. Update the discount and quantity for Greensun product on the boy scout quote with put quote/{id} resource.
  27. Do the verification on Create Multiple Quotes and earn reward points.
  28. Create a contract for ‘Boy Scouts of the Southwest’ with the post on account/contract resource and make note of the generated contract number as we need to pass it in the order request.
  29. Create a contract for ‘Saito Family Laundromats’ with the post on account/contract and make note of the generated contract number.
  30. Create an order for Boyscout account for the "BSofSW - Ground-mounted Panels" opportunity with th post on order resource by passing the contract number and opportunity name.. Make note of the generated order number as we need to pass in the order product request.
  31. Create an order for Laundromat account for the "Saito - Ground-mounted Panels" opportunity with the post on order resource by passing the contract number and opportunity name.
  32. Add Greensun to the Boyscout order with the post on order/product resource by passing the generated order number.
  33. Add the Sunpower and Greensun to the Laundromat order with the post on order/product resource by passing the generated order number.
  34. Remove the Greensun from the Laundromat order with the post on order/removeproduct resource.
  35. Do the verification on Create an Order with Products and earn reward points.
  36. With this we should have 6 green checks as there are 6 trail steps on the Trail: Manage Products, Prices, Quotes, and Orders trail.
  37. Earlier, we have seen how to clean up pricebook and product entries, and now we have to clean lead and account related objects.
  38. Retrieve the accountid for Boyscout and Laundromat account with the get on account resource and delete all the account related objects with the post on deleteaccountcontact resource.
  39. Retrieve the lead id for John Doe and Jane Doe with the get on lead resource and delete both John and Jane with the post on deleteleads resource.
  40. Thanks, Trailhead
